What it is

Three independent open-source subprojects glued by a single prompt maestro. No installer. No lock-in. Cloning the repo and pasting one prompt into opencode reconfigures your entire agent.

Skill Matrix

250 SKILL.md folders covering Next.js, K8s, Go, Rust, ML, design patterns, auto-healing, and more. Auto-discovered on startup.

second-termux

MCP server that runs long jobs in the background and returns a single premium line. ✨ build ✓ 0 (4m12s) @14:30 instead of 400 lines of webpack noise.

engram + zerotoken

Persistent memory between sessions. SQLite + FTS5 with 8 memory types, automatic decay, and 2-3 relevant memories injected per prompt.

How it works

The bundle is configured by a single global opencode.jsonc that points at the repo's subfolders. No copying. No installation scripts. The agent discovers everything at startup.

// ~/.config/opencode/opencode.jsonc
{
  "skills": {
    "paths": [
      "/path/to/My.OpenCode.Conf/skill-matrix/skills-matrix",
      "/path/to/My.OpenCode.Conf/second-termux-v2/opencode-integration"
    ]
  },
  "mcp": {
    "context7": { "type": "remote", "url": "https://mcp.context7.com/mcp", "enabled": true },
    "sqlite":   { "type": "local",  "command": ["npx","-y","mcp-sqlite","./data.db"], "enabled": true },
    "second-termux": { "type": "local", "command": ["node",".../dist/src/server.js"], "enabled": true }
  },
  "plugin": [[ "file:///.../engram+zerotoken/src/engram.ts", { "auto_save": {}, "injection": {} }]]
}
